## About The Project

[![Product Name Screen Shot][product-screenshot]](https://example.com)
This plugin implements a meta-model used to create variable product composition diagrams using the SysML block definition diagrams.
The plugin also implements a new view called "Impact tool" which use the information provided through the use of the meta-model to check constraints, goals, requirements and more.
To use the impact tool, one needs to follow the rules set out to ensure that the assumption made by the tool still hold. 

For more information on how the tool can be used see [Link to thesis](http://example.com/)
